# Projeto Loot

Repositório do **Gênesis** — jogo phygital de coleção, fusão e duelos (criaturas, strains, cartas).

O app Next.js fica na pasta `projeto-loot`. Para desenvolver ou rodar:

```bash
cd projeto-loot
```

Depois: `npm install`, configure `.env.local` (Supabase) e use `npm run dev`. Documentação completa em [projeto-loot/README.md](projeto-loot/README.md).

---

**Scripts na raiz** (fazem `cd projeto-loot` por você):

| Comando       | Descrição              |
|---------------|------------------------|
| `npm run dev` | Servidor de desenvolvimento |
| `npm run build` | Build de produção    |
| `npm run start` | Inicia o app (após build) |
